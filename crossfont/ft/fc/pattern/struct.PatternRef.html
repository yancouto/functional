pub fn add_charset(&self, charset: &CharSetRef) -> bool

Add charset to the pattern.

The referenced charset is copied by Fontconfig internally using FcValueSave so that no references to application provided memory are retained. That is, the CharSet can be safely dropped immediately after being added to the pattern.
